# Artistry: Virtual Art Gallery

**Artistry** is a high-end, immersive digital platform designed to bring world-class art masterpieces directly to users through a curated virtual experience. The application focuses on high-fidelity visual presentation, interactive exploration, and a seamless, gallery-like user interface.

---

## üåü Key Features

* **Interactive Artwork Viewing**: Users can zoom and pan across high-resolution masterpieces to discover intricate details.
* **Dynamic Theme System**: Includes a sophisticated theme toggle supporting both a sleek "Dark Mode" (Default) and a clean "Light Mode" for different viewing preferences.
* **Curated Collections & Exhibitions**: Features dedicated sections for current and upcoming exhibitions, as well as detailed artist profiles.
* **Personalization**: Interactive controls allowing users to "Favorite" artworks, share them, or download them for offline viewing.
* **Responsive & Immersive UI**: Built with a "mobile-first" approach, featuring glassmorphism effects, floating particles, and smooth scroll animations.
* **Rich Metadata**: Every artwork includes detailed information regarding its medium, dimensions, style, and historical location.

### Installation
1.  Clone the repository:
    ```bash
    git clone [https://github.com/your-username/virtual-art-gallery.git](https://github.com/your-username/virtual-art-gallery.git)
    ```
2.  Navigate to the project directory:
    ```bash
    cd virtual-art-gallery
    ```
3.  Open `index.html` in your preferred browser:
    ```bash
    # Example for MacOS
    open index.html
    ```

## üì∏ Interface Overview

The application is structured into several immersive sections:
* **Welcome**: A cinematic hero section with floating particle effects.
* **Gallery**: The core interactive hub with a main display, navigation controls, and a thumbnail browser.
* **Exhibitions**: A grid-based layout showcasing current and upcoming art events.
* **Artists**: Biographies and specialty details for featured creators.
* **About & Contact**: Historical context of the gallery and a functional contact form for inquiries.
